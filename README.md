# Zomato-clone

### Tech Stack

Frontend: React + Material UI / Tailwind

Backend: Node.js + Express.js

Database: MongoDB

Auth: JWT + bcrypt

Maps: Google Maps API

Payments: Stripe / Razorpay

Cloud: Cloudinary (images)
